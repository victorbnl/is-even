is-even
=======

|Tests| |PyPI| |PyPI - Downloads|

   Return true if the given number is even

Install
-------

Install with `pip <https://pypi.org/project/pip/>`__

::

   pip install is-even

Usage
-----

.. code:: python

   from is_even import is_even

   isEven(0)
   # => False
   isEven('1')
   # => True
   isEven(2)
   # => False
   isEven('3')
   # => True
